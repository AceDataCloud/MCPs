# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2026.4.25.1] - 2026-04-25

### Added
- Initial release of MCP OpenAI server
- Chat completions with GPT-4, GPT-4o, GPT-5, and o-series models
- Responses API with extended model variant support
- Image generation with gpt-image-1, gpt-image-1.5, gpt-image-2, dall-e-3, and nano-banana models
- Image editing with reference image support
- Text embeddings with text-embedding-3-small, text-embedding-3-large, and ada-002
- Informational tools for listing available models
