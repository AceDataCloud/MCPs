# OpenAI MCP for VS Code

OpenAI API via AceDataCloud - Chat Completions, Embeddings, Image Generation, Responses, and Image Editing.

## Setup

1. Get your API token from [AceDataCloud Platform](https://platform.acedata.cloud)
2. Set `ACEDATACLOUD_API_TOKEN` in the extension settings

## Tools

- `openai_chat_completion` - Chat completions
- `openai_embeddings` - Text embeddings
- `openai_image_generate` - Image generation
- `openai_responses` - Responses API
- `openai_image_edit` - Image editing
- `openai_list_models` - List available models
